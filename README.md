# Maersk-Logistics-Analytics-Dashboard-Power-BI-Project**

**Description:**

Developed an interactive Power BI dashboard to analyze logistics operations for Maersk . Implemented data cleaning, transformation, and modeling using Power Query and DAX. Delivered actionable insights on time delivery rates, customer satisfaction (CSAT), and operational bottlenecks. .

**🎯 Key Features****
KPI tracking: Total Orders, On-Time Delivery %, CSAT, Avg Delivery Time
Hub utilization and capacity analysis
Driver performance and experience insights
Vehicle usage and maintenance tracking
Drill-through pages for detailed analysis
🛠 Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Power Query (ETL)
Data Modeling

**📈 Insights Generated**

Identified underutilized hubs
Evaluated driver performance vs experience
Monitored delivery efficiency trends
Highlighted vehicle usage patterns

**📂 Files Included**

Power BI Dashboard (.pbix)
Dataset files (.csv)

**🚀 Use Case**

This project simulates real-world logistics analytics and demonstrates business intelligence skills applicable to supply chain and operations roles.
